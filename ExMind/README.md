# ExMind - 엑셀 기능이 보강된 마인드맵 프로그램

macOS와 iOS에서 동작하는 크로스 플랫폼 마인드맵 애플리케이션입니다.

## 주요 기능

### 🎨 세 가지 뷰 모드

1. **캔버스 모드** (Canvas Mode)
   - 시각적 마인드맵 뷰
   - `[타이틀]:값` 형태로 노드 표시
   - 드래그 앤 드롭으로 노드 이동
   - 곡선 연결선으로 계층 구조 표시
   - 줌 인/아웃 지원
   - 더블클릭으로 제목 및 노트 편집
   - 수식 오류 시 취소선으로 표시

2. **테이블 모드** (Table Mode)
   - 엑셀 스타일의 테이블 뷰
   - 모든 노드를 행으로 표시
   - 수식 입력과 계산 결과를 함께 표시
   - 커스텀 컬럼 추가 가능
   - 인라인 편집 지원
   - 수평/수직 스크롤 지원

3. **아웃라인 모드** (Outline Mode)
   - 계층 구조를 트리 형태로 표시
   - `[타이틀]:값` 형태로 표시
   - 접기/펼치기 기능
   - 빠른 네비게이션

### 🧮 엑셀 스타일 수식 계산

- **숫자 입력**: `100` → 100으로 표시
- **수식 입력**: 
  - `=50+50` → 100으로 계산
  - `=대전+서울` → 다른 노드 값 참조
  - `=100*2+50` → 사칙연산 지원
  - `=(100+50)/2` → 괄호 지원
- **수식 오류 처리**: 잘못된 수식은 취소선으로 표시
- **실시간 계산**: 참조 노드 변경 시 자동 재계산

#### 수식 예시
```
노드 구조:
[노드] : 120         (수식: =대전+서울)
  ├─ [대전] : 100    (수식: 100)
  └─ [서울] : 20     (수식: 20)
```

### 📊 엑셀 스타일 속성 시스템

- 각 노드에 커스텀 속성 추가 가능
- 다양한 데이터 타입 지원:
  - 텍스트 (Text)
  - 숫자 (Number)
  - 날짜 (Date)
  - 불린 (Boolean)
- 테이블 뷰에서 엑셀처럼 데이터 편집
- 속성 패널에서 상세 정보 관리

### 🎯 핵심 기능

- **계층 구조**: 부모-자식 관계로 노드 구성
- **색상 커스터마이징**: 10가지 프리셋 색상 제공
- **자동 저장**: SwiftData를 사용한 자동 데이터 저장
- **멀티플랫폼**: macOS와 iOS에서 동일한 기능 제공
- **수식 계산**: 엑셀처럼 수식을 사용한 자동 계산

## 사용 방법

### 문서 관리

1. **새 마인드맵 만들기**: 좌측 사이드바에서 `+` 버튼 클릭
2. **문서 선택**: 목록에서 원하는 문서 클릭
3. **문서 삭제**: 스와이프하여 삭제 버튼 표시 (iOS) 또는 우클릭 (macOS)

### 노드 작업

1. **노드 추가**: 
   - 루트 노드: 상단 툴바의 "Add Node" 버튼
   - 자식 노드: 노드 선택 후 `+` 아이콘 클릭

2. **노드 편집**:
   - 캔버스 모드: 더블클릭하여 제목 또는 노트 편집
   - 테이블 모드: 셀을 클릭하여 직접 편집
   - 아웃라인 모드: 클릭하여 편집

3. **노트(수식) 입력**:
   - 숫자: `100` (그대로 표시)
   - 수식: `=50+50` (계산 결과 표시)
   - 참조: `=대전+서울` (다른 노드 값 사용)
   - 복잡한 수식: `=(100+50)*2/3`
   - **노트**: 노드 타이틀로 다른 노드를 참조할 수 있습니다

4. **노드 이동** (캔버스 모드):
   - 노드를 드래그하여 원하는 위치로 이동

5. **노드 삭제**:
   - 상단 툴바의 "Delete Node" 버튼
   - 선택된 노드의 쓰레기통 아이콘
   - macOS: Delete 키 또는 Backspace 키

6. **노드 색상 변경**:
   - 노드 선택 후 하단 속성 패널에서 색상 선택

### 수식 사용 예시

```
예시 1: 간단한 계산
[총합] : 120
  수식: =100+20

예시 2: 노드 참조
[전체매출] : 500
  수식: =서울+부산+대전
  ├─ [서울] : 200
  │   수식: 200
  ├─ [부산] : 150
  │   수식: 150
  └─ [대전] : 150
      수식: 150

예시 3: 복잡한 수식
[평균] : 150
  수식: =(서울+부산+대전)/3

예시 4: 오류 표시
[잘못된수식] : ~~=서울+존재하지않음~~
  수식: =서울+존재하지않음 (취소선 표시)
```

### 커스텀 속성 추가

1. 노드 선택
2. 하단 속성 패널에서 `+` 버튼 클릭
3. 속성 이름과 타입 선택
4. 값 입력

## 기술 스택

- **SwiftUI**: 사용자 인터페이스
- **SwiftData**: 데이터 저장 및 관리
- **Swift 6**: 최신 Swift 언어 기능 활용
- **FormulaEngine**: 커스텀 수식 계산 엔진
- **크로스 플랫폼**: iOS 18+ / macOS 15+
